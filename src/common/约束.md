# common/ 共享层编码约束

## 1. 目录职责

- `utils/` — 双方共用的工具函数（request.js、user.js、validate.js、code.js 等）
- `filters/` — 全局 Vue 过滤器
- 本目录只放纯工具函数，不包含组件、样式、常量

## 2. 导入规则

- `common/` 下的模块可以被 `admin/`、`business/` 和顶层文件引用
- `common/` 下的模块不得引用 `admin/` 或 `business/` 域下的模块
- `common/` 可以引用第三方库（Vue、Axios 等）

## 注释要求

- 代码应添加适量注释，关键逻辑、复杂算法、接口定义、配置项等必须有注释说明

## 错误处理

- 报错提示、日志信息、异常消息等使用中文编写，尽量减少英文使用
