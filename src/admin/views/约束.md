# admin/views/ 后台页面视图编码约束

## 1. 目录规范

- 按功能域组织：`resource/`、`system/`、`platform/`、`personal/`、`common/`
- 每个页面一个目录，kebab-case 命名，入口为 `index.vue`
- 嵌套子页面放在父目录下

## 2. 域隔离

- 本目录存放后台管理页面，不得被 `business/` 域引用
- 前台页面放在 `business/views/`
- 双方共用页面（如登录页）各自维护一份，不共用

## 注释要求

- 代码应添加适量注释，关键逻辑、复杂算法、接口定义、配置项等必须有注释说明

## 错误处理

- 报错提示、日志信息、异常消息等使用中文编写，尽量减少英文使用
