# admin/styles/ 后台样式编码约束

## 1. 文件组织

- `variables.scss` — 全局变量定义
- `mixin.scss` — 可复用 mixin
- `index.scss` — 样式入口，导入所有模块
- `element-variables.scss` — Element UI 主题变量覆盖
- `sidebar.scss` — 侧边栏样式
- `transition.scss` — 过渡动画
- `flex.scss` — Flexbox 工具类

## 2. 域隔离

- 本目录为后台管理专用样式，仅由 `admin/layout/` 和 `admin/views/` 使用
- 前台样式放在 `business/styles/`，互不干扰

## 注释要求

- 代码应添加适量注释，关键逻辑、复杂算法、接口定义、配置项等必须有注释说明

## 错误处理

- 报错提示、日志信息、异常消息等使用中文编写，尽量减少英文使用
