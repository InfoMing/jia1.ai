# admin/layout/ 后台布局组件编码约束

## 1. 组件结构

- `index.vue` — 主布局入口
- `components/Navbar.vue` — 顶部导航栏
- `components/Sidebar/` — 侧边栏
- `components/AppMain.vue` — 主内容区
- `components/TagsView/` — 标签页导航
- `components/Settings/` — 设置面板

## 2. 规范

- 监听窗口 resize 事件，通过 mixin 管理
- 侧边栏根据路由动态生成
- 支持移动端响应式适配
- 使用 `keep-alive` 缓存已访问页面

## 3. 域隔离

- 本目录为后台管理专用布局，不得被 `business/` 域引用
- 前台布局放在 `business/layout/`

## 注释要求

- 代码应添加适量注释，关键逻辑、复杂算法、接口定义、配置项等必须有注释说明

## 错误处理

- 报错提示、日志信息、异常消息等使用中文编写，尽量减少英文使用
