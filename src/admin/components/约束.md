# src/components 组件编码约束

## 1. 目录规范

- 每个组件一个目录，PascalCase 命名
- 入口文件为 `index.vue`
- 子组件放在同一目录下

## 2. 组件规范

- `name` 属性与目录名一致（PascalCase）
- Props 使用对象形式，必须指定类型
- 对象/数组默认值使用工厂函数
- 回调方法通过 Function prop 传入
- 使用命名插槽：`<slot name="buttonSlot" />`、`<slot name="columnButton" :row="scope.row">`
- 使用 `scoped` 样式隔离

## 3. BaseTable 组件模式

- 统一使用 `<base-table>` 处理列表页 CRUD
- 通过 prop 注入 API、列定义、搜索字段、按钮配置
- 插槽：`buttonSlot`（操作按钮）、`heightSearchSlot`（高级搜索）、`editOrAddInputSlot`（编辑表单）

## 注释要求

- 代码应添加适量注释，关键逻辑、复杂算法、接口定义、配置项等必须有注释说明

## 错误处理

- 报错提示、日志信息、异常消息等使用中文编写，尽量减少英文使用
