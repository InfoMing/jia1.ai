# BaseTable 通用表格组件

- 封装 Element UI el-table
- Props：api、columns、headerTitle、showPage、pageLayout、addObj、list、deleteObj 等
- 插槽：buttonSlot、heightSearchSlot、editOrAddInputSlot、columnButton
- 通过 Function prop 实现回调，不 $emit 自定义事件

## 注释要求

- 代码应添加适量注释，关键逻辑、复杂算法、接口定义、配置项等必须有注释说明

## 错误处理

- 报错提示、日志信息、异常消息等使用中文编写，尽量减少英文使用
