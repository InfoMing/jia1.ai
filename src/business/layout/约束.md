# views/business/layout/ 业务布局编码约束

## 1. 导航栏（Navbar）

### 1.1 登录态判断

```js
checkLogin() {
  const user = sessionStorage.getItem('LOCAL_CACHE_USER_INFO_KEY')
  this.isLoggedIn = !!user
}
```

- 在 `created()` 和 `$route` watch 中调用 `checkLogin()`
- 登录后显示用户头像 + 名称 + 下拉菜单
- 未登录显示「登录」「注册」按钮

### 1.2 导航链接

| 链接 | 路径 | 说明 |
|---|---|---|
| 首页 | `/business/market` | logo 点击 + 首页导航项 |
| 市场行情 | `/business/market/a-share` | |
| 港股 | `/business/market/hk` | |
| 美股 | `/business/market/us` | |
| 黄金 | `/business/market/gold` | |

### 1.3 用户下拉菜单

已登录用户显示下拉菜单，菜单项：

1. 我的监控 → `/business/monitor`
2. 我的关注 → `/business/watchlist`
3. 触发记录 → `/business/trigger-record`
4. 短信记录 → `/business/sms-record`
5. 设置 → `/business/settings`
6. 退出登录 → 清空缓存 + `location.reload()`

### 1.4 退出登录

```js
handleLogout() {
  sessionStorage.clear()
  window.localStorage.clear()
  location.reload()
}
```

### 1.5 移动端响应式

- 768px 以下导航项隐藏，显示汉堡菜单按钮
- 汉堡菜单展开所有导航项 + 用户相关菜单项

## 2. 页脚（Footer）

- 固定底部，不随页面滚动
- 链接：关于我们、帮助中心、免责声明、用户协议
