# 首页资源清单

新首页只从本目录导入资源，不依赖 `public/legacy`、旧 `/static` 或线上静态文件。

## shared

- `mascot.png`：Super-i 导航品牌角色，来源于既有线上首页原素材。

## home

- `media/hero.mp4`、`hero-poster.jpg`：首页 Sticky Hero 视频与封面。
- `courses/course-*.webp`：2026-07-10 固化的四张精选课程封面。
- `showcase.jpg`、`showcase-mark.svg`：品牌宣传片模块。
- `community-qr.jpg`、`community-panel.png`：社群二维码及展示图。
- `career.png`、`icons/*`：岗位与职业定位模块原素材。

素材仅用于本项目既有页面重构。页面中的线上地址只保留为尚未重构课程详情页的导航目标。
