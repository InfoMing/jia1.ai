# 前台资源清单

- `shared/logo-mark.png`：公共导航 Logo。
- `shared/mascot.png`：网站 favicon。
- `home/media/hero.mp4`、`home/hero-poster.jpg`：首页视频首屏。
- `home/showcase.jpg`、`home/showcase-mark.svg`：品牌宣传片模块。
- `home/community-qr.jpg`、`home/community-panel.png`：社群模块。
- `home/career.png` 与 `home/icons/audience-*.svg`：岗位和职业定位模块。

资源只能由 Vue 源码本地引用，不得依赖旧 `public` 目录或线上静态地址。
